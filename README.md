# BolsaExpert🎯

Um quiz interativo para adivinhar empresas a partir do ticker da bolsa de valores brasileira.

## Como jogar

1. Digite seu nome;
2. Responda 10 perguntas com tempo limitado;
3. Veja sua pontuação e ranking.
